Boolean Order of Operations

The priority between "and" and "or" can be determined by using two boolean statements.

            >>> 1 or 0 and 0
            1
            >>> (1 or 0) and 0
            0

If "or" had a higher priority than "and" then adding parenthesis around the arguments to "or" would have no effect. However, if "and" had a higher priority the parenthesis would change the outcome. Since the later is the observed outcome, "and" has a higher priority than "or".

We can also determine the priority between "not" and "and" by using two very similar boolean statements.

            >>> not 0 and 0 or 0
            0
            >>> not (0 and 0) or 0
            1

We can use the same logic as above to determine that "not" has a higher priority than "and". If "and" had a higher priority than "not" then adding parenthesis would not have changed the outcome.
